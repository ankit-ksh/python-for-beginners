# Research Papers & Primary Sources

Authoritative references about Python’s design, style, and evolution. These are great for deeper understanding and teaching.

## PEPs (Python Enhancement Proposals)
- PEP 20 – The Zen of Python
	- https://peps.python.org/pep-0020/
- PEP 8 – Style Guide for Python Code
	- https://peps.python.org/pep-0008/
- PEP 257 – Docstring Conventions
	- https://peps.python.org/pep-0257/
- PEP 484 – Type Hints
	- https://peps.python.org/pep-0484/
- PEP 572 – Assignment Expressions (the “walrus” operator)
	- https://peps.python.org/pep-0572/
- PEP 622 / Structural Pattern Matching overview (Python 3.10)
	- https://peps.python.org/pep-0636/
