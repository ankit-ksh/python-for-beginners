# Inspiring Projects Built With Python

A curated set of real, widely‑used projects written in Python. Skim a few, read their READMEs, and peek at code to get inspired.

## Command‑line powerhouses
- yt‑dlp — feature‑rich YouTube/media downloader (active fork of youtube‑dl)
	- https://github.com/yt-dlp/yt-dlp
- httpie — human‑friendly HTTP client
	- https://httpie.io/
- cookiecutter — project scaffolding from templates
	- https://cookiecutter.readthedocs.io/
- poetry — modern dependency & packaging tool
	- https://python-poetry.org/
- pipx — install and run Python apps in isolated envs
	- https://pipx.pypa.io/
- black — opinionated code formatter
	- https://black.readthedocs.io/
- ruff — ultra‑fast Python linter
	- https://docs.astral.sh/ruff/

## Data, science, and notebooks
- Jupyter / IPython — interactive computing and notebooks
	- https://jupyter.org/
- pandas — data analysis with DataFrames
	- https://pandas.pydata.org/
- scikit‑learn — machine learning toolkit
	- https://scikit-learn.org/
- Scrapy — fast and powerful web scraping framework
	- https://scrapy.org/

## Web & APIs
- Django — batteries‑included web framework
	- https://www.djangoproject.com/
- FastAPI — modern, type‑hinted APIs, great DX
	- https://fastapi.tiangolo.com/
- Flask — minimal web microframework
	- https://flask.palletsprojects.com/
- MkDocs — static site generator geared towards docs
	- https://www.mkdocs.org/

## Automation, ops, and data pipelines
- Apache Airflow — programmatic workflows (DAGs)
	- https://airflow.apache.org/
- Ansible — IT automation and configuration management
	- https://www.ansible.com/
- Home Assistant — open‑source home automation hub
	- https://www.home-assistant.io/

## Hardware, games, and creative
- MicroPython — Python for microcontrollers
	- https://micropython.org/
- Pygame — game development set of modules
	- https://www.pygame.org/
- Manim — mathematical animation engine
	- https://www.manim.community/

## Security & networking
- mitmproxy — interactive HTTPS proxy
	- https://mitmproxy.org/
- Paramiko — SSH for Python
	- https://www.paramiko.org/

## Learning more collections
- Awesome Python — massive curated index of Python projects
	- https://github.com/vinta/awesome-python
- Trending Python on GitHub — see what’s hot now
	- https://github.com/trending/python

How to explore
- Read each project’s README and docs.
- Try small examples locally; skim source files for structure and style.
- Notice packaging, tests, CI, and contribution guidelines for real‑world practices.
