# Tools

Practical tools you’ll use day‑to‑day with Python. Prefer official docs and well‑maintained projects.

## Editors & extensions
- VS Code Python extension (Microsoft)
	- https://marketplace.visualstudio.com/items?itemName=ms-python.python
	- Docs: https://code.visualstudio.com/docs/languages/python
- PyCharm (Community Edition is free)
	- https://www.jetbrains.com/pycharm/

## Environments & packaging
- venv – built‑in virtual environments
	- https://docs.python.org/3/library/venv.html
- pip – package installer
	- https://pip.pypa.io/
- pipx – install and run Python apps in isolated envs
	- https://pipx.pypa.io/
- Poetry – dependency management and packaging
	- https://python-poetry.org/
- Hatch – modern project manager
	- https://hatch.pypa.io/
- Conda / Miniconda – cross‑language env & packages
	- https://docs.conda.io/

## Notebooks & data
- JupyterLab / Notebook
	- https://jupyter.org/
- VS Code Jupyter extension
	- https://marketplace.visualstudio.com/items?itemName=ms-toolsai.jupyter

## Quality & automation
- Black – code formatter
	- https://black.readthedocs.io/
- Ruff – fast linter
	- https://docs.astral.sh/ruff/
- mypy – static type checker
	- https://mypy.readthedocs.io/
- pre-commit – manage git hook checks
	- https://pre-commit.com/
- pytest – testing framework
	- https://docs.pytest.org/

## Version control
- Git – official docs
	- https://git-scm.com/doc
- GitHub – guides
	- https://docs.github.com/en/get-started/start-your-journey

## Useful CLIs
- HTTPie – user‑friendly HTTP client
	- https://httpie.io/
- `uv` – fast Python package installer and runner
	- https://docs.astral.sh/uv/
