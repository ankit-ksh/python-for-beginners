# Interactive Playgrounds

- Trinket (Python 3)
	- https://trinket.io/python3
	- In‑browser Python with input() support; easy embeds for lessons.

- Replit (Python)
	- https://replit.com/languages/python3
	- Full online IDE with file/project support; great for quick sharing.

- Python Tutor (Visualize execution)
	- https://pythontutor.com/
	- Step through code line‑by‑line; excellent for learning concepts.

- Google Colab (Notebooks)
	- https://colab.research.google.com/
	- Free GPU/TPU tiers for data/science notebooks; easy sharing.

- JupyterLite (In‑browser Jupyter via WebAssembly)
	- https://jupyterlite.readthedocs.io/en/latest/try/lab/
	- No server required; runs entirely in the browser.

- Kaggle Notebooks
	- https://www.kaggle.com/code
	- Free hosted notebooks with datasets; great for data practice.
