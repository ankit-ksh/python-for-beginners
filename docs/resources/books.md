# Books

- Automate the Boring Stuff with Python (2e) — Al Sweigart
	- https://automatetheboringstuff.com/
	- Free to read online. Practical, project‑based intro for absolute beginners.

- Think Python (2e) — Allen B. Downey
	- https://greenteapress.com/wp/think-python-2e/
	- Free textbook focused on fundamentals and problem‑solving.

- Python Crash Course (2e) — Eric Matthes
	- https://nostarch.com/pythoncrashcourse2e
	- Clear, hands‑on intro with projects (games, data visualization, web apps).

- Effective Python (2e) — Brett Slatkin
	- https://effectivepython.com/
	- 90 actionable items to write better Python. Great after a first course.

- Fluent Python (2e) — Luciano Ramalho
	- https://www.oreilly.com/library/view/fluent-python-2nd/9781492056348/
	- Deep dive into Pythonic patterns (data model, typing, async, concurrency).

- Introduction to Computation and Programming Using Python (3e) — John Guttag
	- https://mitpress.mit.edu/9780262542364/
	- Rigorous CS‑flavored intro (computation, modeling, data analysis).

- Dive Into Python 3 — Mark Pilgrim
	- https://diveintopython3.net/
	- Free online. Slightly older but still useful for language internals.
